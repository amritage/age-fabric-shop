# Age Fabric Shop Backend

## Setup

1. Clone the repo
2. Run `npm install`
3. Create `.env` file (see `.env.example`)
4. Run server: `npm run dev`

## Features

- Secure Auth
- Admin Panel
- Cloudinary upload

## Prerequisites

- Node.js (v14 or higher)
- MongoDB

## Environment Variables

- See `.env.example` for required variables.

## Testing

- Run `npm test` to execute tests with Jest.
